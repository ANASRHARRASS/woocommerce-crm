== Description ==
The Universal Lead Capture Plugin is a powerful tool designed to help you capture leads from various platforms and integrate them seamlessly into your workflow. This plugin supports integration with popular platforms such as WooCommerce, Elementor, HubSpot, Zoho, Google Ads, and Facebook Ads, allowing you to create dynamic forms that can capture valuable customer information.

== Installation ==
1. Upload the `universal-lead-capture-plugin` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Configure the plugin settings under the 'Universal Lead Capture' menu in the WordPress admin.

== Usage ==
Once activated, you can create and manage forms directly from the plugin settings. The plugin allows you to integrate with WooCommerce to include product and shipment functionalities in your forms. You can also connect with HubSpot and Zoho to organize and manage your leads effectively.
